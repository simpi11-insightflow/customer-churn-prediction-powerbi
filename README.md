CUSTOMER CHURN PREDICTION & RETENTION OPTIMIZATION
This project focuses on analyzing customer churn behavior and identifying the major factors affecting customer retention using SQL,Pyhton,Machine Learning and Power BI. The main goal of this project is to understand customer behavior, identify high-risk churn segments, and generate business insights that can help improve customer retention strategies.

PROJECT OVERVIEW
In this project, I performed :
- Data cleaning and preprocessing using SQL.
- Exploratory Data Analysis using Python.
- Customer churn prediction using Machine Learning.
- Interactive dashboard creation using Power BI.
- Business impact analysis using customer insights.
The dashboard help analyze churn patterns based on:
- Contract type
- Internet services
- Payment methods
- Customer tenure
- Tech support availability
- Online security
- Streaming services
- Revenue contribution

TOOLS & TECHNOLOGIES USED 
- SQL
- Python
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

KEY INSIGHTS
- Customers with month-to-month contracts showed the highest churn rate.
- Customers without tech support and online security were more likely to leave.
- Early-stage-customers had higher churn probability.
- Premium customers contributed the highest revenue and CLTV.
- Customers using electronic check payment methods showed higher churn behavior.
- Around 22% customers were predicted to churn, indicating a significant retention risk segment.
